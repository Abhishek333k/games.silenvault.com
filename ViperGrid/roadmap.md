# Neon Shatter: Project Strategy

## 🕹️ Core Game Concept
A high-fidelity, arcade-style brick breaker with neon aesthetics, procedural levels, and a "Viper" power-up system.

## 🛠️ Feature Task List

### Phase 1: Foundations
- [ ] Responsive 60FPS Canvas Engine
- [ ] Physics-based Paddle (Mouse/Touch/Keyboard)
- [ ] Ball Reflection Vector Logic (Precision Angle Control)
- [ ] Basic Brick Grid Generator

### Phase 2: The "Grid" Architecture
- [ ] Hardened Bricks (2-5 hits with visual cracking)
- [ ] Explosive "Nitro" Bricks
- [ ] Invisible "Ghost" Bricks
- [ ] Moving Brick Patterns

### Phase 3: The Viper Power-Up System
- [ ] Laser Paddle (Weaponized Brick Breaking)
- [ ] Multi-Ball (Split Logic)
- [ ] Fireball (Penetration Logic)
- [ ] Safety Shield (Bottom protection)
- [ ] Magnet Paddle (Catch and Release)

### Phase 4: Aesthetics & UX
- [ ] Neon Bloom CSS/Canvas Shaders
- [ ] Screen Shake Engine
- [ ] Procedural Synth-wave Audio (Web Audio API)
- [ ] Combo Multiplier System

### Phase 5: Progression & Meta
- [ ] Procedural Level Patterns (infinite levels)
- [ ] Save System (High Scores/Unlocked Skins)
- [ ] Ascension Mode (Difficulty multiplier)
